## What is Reference Documentation?

Reference documentation is **information-oriented** material that describes the machinery of the product. It's organized for easy lookup when you need specific technical details.

## Reference Characteristics

- **Information-focused**: Comprehensive technical details
- **Well-organized**: Easy to scan and find information
- **Accurate**: Precise and up-to-date specifications
- **Complete**: Covers all aspects of the topic
